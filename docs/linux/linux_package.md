## package
### パッケージ管理
Debian系では、dpkgコマンドやaptコマンドを使用してパッケージ管理を行う。  
Red Hat系では、rpmコマンドやyumコマンドやdnfコマンドを使用してパッケージ管理を行う。  
基本的にはDebian系ではaptコマンド、Red Hat系ではdnfコマンドを使用する。

---

### dpkg
```
sudo dpkg [option] <action>
```
