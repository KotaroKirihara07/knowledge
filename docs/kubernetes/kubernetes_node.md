# [Kubernetes](kubernetes_index.md)
## Node
ノードには「コントロールプレーンノード」、「ワーカーノード」の2種類ある。

---

### コントロールプレーンノード
APIの公開、アプリケーションのスケジュール設定、自己修復機能の実装、スケーリング操作の管理などの管理などを行うノード。  
コントロールプレーンは以下の?種類の...
- APIサーバ
- スケジューラー
- コントローラーマネージャー
- コントローラー
  - Deploymentコントローラー
  - StatefulSetコントローラー
  - ReplicaSetコントローラー
- クラスターストア   

---

### ワーカーノード
ビジネスアプリケーションを実行するためのノード。
主要なワーカーノードコンポーネント。
- Kubelet
- Runtime
- Kube-proxy (Network proxy)
