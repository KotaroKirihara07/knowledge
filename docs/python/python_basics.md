### ファイル
```
with open("data/example.txt", mode=<mode>, encoding="UTF-8") as f:
    # 処理
```

|method|description|
|-----|-----|
|f.read()|テキスト内のすべてのデータをひとつの変数に格納する|
|f.readline()|1行ごとのデータ|
|f.readlines()|データを行単位で分割したリストとして格納する|
|f.write("wwww")|指定した文字列を書き込み、書き込まれた文字数を返す|
|f.writelines(list1)|指定した文字列リストを1つずつ書き込む|

|mode|description|
|-----|-----|
|"r"|読み込みモードで開く (デフォルト)|
|"r+"|読み書きモードで開く (デフォルト)|
|"a"|追記モードで開く|
|"a+"|読み込み+追記モードで開く|
|"w"|書き込みモード開く(ファイルが存在しなかった場合は新しく作成)|
|"w+"|読み書きモード開く(ファイルが存在しなかった場合は新しく作成)|
|"x"|書き込みモード開く(ファイルが存在する場合はエラー)|
|"x+"|読み書きモード開く(ファイルが存在する場合はエラー)|
|"b"|バイナリモード|
|"t"|テキストモード|
